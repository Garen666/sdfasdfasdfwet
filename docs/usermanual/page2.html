Создание категории

Создание категории в CMS OneClick Store достаточно простой процесс. Каждая созданная категория имеет свой способ отображения, который может быть задан позже в процессе редактирования. В данном разделе мы рассмотрим создание категорий, вложенных категорий и настройку порядка их отображения.

Создание категорий интернет-магазина, формирование меню категорий

  1. Перейдите в административную панель в раздел Настройки => Навигация => Категории
  2. Чтобы добавить категорию, нажмите вкладку "Добавить категорию"
  3. Укажите имя категории и нажмите "Вставить новую запись"
  4. Создайте таким же способом еще несколько категорий

Настройка порядка отображения категорий на пользовательской части сайта

  1. Перейдите в административную панель в раздел Настройки => Навигация => Категории
  2. В таблице существующих категорий найдите столбец "Порядок сортировки"
  3. Кликните мышкой на значении в столбце для нужной категории
  4. Введите числовое значение руководствуясь правилом: меньшее число - начало, большее число- конец списка

Добавление вложенных категорий, формирование выпадающего меню категорий

  1. Перейдите в административную панель в раздел Настройки => Навигация => Категории
  2. Нажмите вкладку "Добавить категорию"
  3. Укажите имя категории
  4. В выпадающем меню "Родительская категория" выберите категорию, в которую будет вложена создаваемая
  5. Нажмите "Вставить новую запись"
