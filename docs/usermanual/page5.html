Создание текстовой страницы

Помимо категорий с товарами CMS OneClick Store поддерживает создание текстовых страниц. Эта функция может быть использована для создания статей, обзоров товаров и так далее. К тому же, для каждой страницы можно настроить свою логику работы. Все созданные страницы формируют собой меню в хеддере сайта. Давайте рассмотрим создание страниц пошагово.

Создание простой текстовой страницы

  1. Перейдите в административную панель в раздел Настройки => Навигация => Страницы
  2. Чтобы создать страницу нажмите вкладку "Добавить новую страницу"
  3. В поле "Название страницы" введите название Вашей будущей страницы
  4. В поле "Название кнопки введите текст название пункта меню, отвечающего за переход на создаваемую страницу
  5. Поле "Содержание страницы" содержит мощный текстовый редактор, который позволит вам создать красочное содержимое Вашей станицы
  6. Нажмите кнопку "Вставить новую запись"


Создание вложенной страницы c логикой

  1. Перейдите в административную панель в раздел Настройки => Навигация => Страницы
  2. Чтобы создать страницу нажмите вкладку "Добавить новую страницу"
  3. В поле "Название страницы" введите название Вашей будущей страницы
  4. В поле "Название кнопки введите текст название пункта меню, отвечающего за переход на создаваемую страницу
  5. Поле "Содержание страницы" содержит мощный текстовый редактор, который позволит вам создать красочное содержимое Вашей станицы
  6. В выпадающем меню "Родительская страница" выберите страницу, относительно которой создаваемая страница будет вложенной
  7. В выпадающем меню "Тип (логика) страницы выберите "Отзывы о магазине (guestbook)"
  8. Нажмите кнопку "Вставить новую запись"
