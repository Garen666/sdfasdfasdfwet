Создание фильтров\характеристик и назначение их товарам


Интернет-магазины на базе CMS OneClick Store обладают мощной и гибкой в настройках системой фильтров и характеристик. Так, для каждой категории товаров есть возможность задать собственные фильтры товаров нескольких типов (cheсkbox, radiobox, select, interval), которые отображаются слева от основного блока контента. А у каждого товара может быть своя табличка характеристик со значениями, которые также настраиваются индивидуально.


Каждый отдельный фильтр, по сути, состоит из двух частей: название и значение. Таким образом, вы можете заранее создать неограниченное количество необходимых для вас фильтров\характеристик, после чего присваивать их необходимым товарам и вписывать значение для каждого конкретного случая. К примеру, фильтр "Диагональ" будет общим для всей категории "Мониторы", а его значение можно указать для каждого товара отдельно.


Каждому товару можно присвоить до 10 отдельных фильтров. Во время присвоения, администратору доступны две опции, которые регулируют поведение фильтра\характеристики: Доступен в фильтрах и Характеристика в таблице. Каждая опция включается чек-боксом.
