CommentsAPI - небольшой пакет, который реализует самый базовый функционал
системы комментариев.

Пакет сделан так, что его использование возможно только инкапсуляцией
(аггрегацией) CommentsAPI в свой проект.

Например, после подключения пакета CommentsAPI можно написать:

// добавить комментарий
CommentsAPI::Get()->addComment(...);

// получить все комментарии по некому ключу
$comments = CommentsAPI::Get()->getComments($key);

// и т.д. есть другие методы

==============================
Что такое ключ (key)?
==============================
Когда вы добавляете комментарий, то он привязывается к некому ключу
(строка 255 символов). Например, "news-100500", означает что комментарий
привязан к новости с id=100500. Разработчик сам определяет "ключи",
добавляет комментарии, а затем получает их по этому ключу.
